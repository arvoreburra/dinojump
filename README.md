# Dino Jump
Jogo de plataforma casual estilo cartoon onde você controla um dinossauro subindo infinitamente pelas nuvens. Desenvolvido como projeto acadêmico.

Dino Jump é um endless platformer inspirado no clássico Doodle Jump e no Pou. O objetivo é simples: suba o máximo que puder sem cair! Quanto mais alto você chega, mais pontos acumula — mas cuidado com as nuvens que se movem e as que quebram ao ser pisadas.

# Diferenciais
Controle por giroscópio no celular;

Leaderboard global em tempo real via Firebase;

Delta Time — velocidade idêntica em qualquer monitor (60Hz, 144Hz...);

Estilo cartoon com animações e partículas;
